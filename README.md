Builder)
📌 Опис проєкту
Це інтелектуальна система для проєктування, симуляції та оптимізації захисних споруд (баз) за мотивами гри Rust. Проєкт поєднує в собі процедурну генерацію за допомогою генетичних алгоритмів та навчання з підкріпленням (Reinforcement Learning) для створення максимально ефективних архітектурних рішень.

🚀 Ключові можливості
Reinforcement Learning (RL): Використання Deep Q-Network (DQN) з мульти-дискретним простором дій для навчання агента будувати бази з нуля.

Генетичні алгоритми (GA): Процедурна генерація споруд з використанням турнірної селекції, елітизму та адаптивної мутації для пошуку оптимальних планувань.

Фізичний рушій та колізії: Кастомна реалізація просторового хешування (Spatial Hashing) для швидкої перевірки колізій та розрахунок структурної стабільності в реальному часі.

Графовий аналіз: Побудова 3D-графа споруди для оцінки логістики (алгоритм Дейкстри) та розрахунку стійкості до рейдів (враховуючи splash-урон від вибухів).

Інтерактивний UI: GUI на JavaFX з підтримкою камери (zoom/pan), системою "привидів" для будівництва та візуалізацією стабільності блоків.

🛠 Технологічний стек
Language: Java 17+

Frameworks: JavaFX (UI), Maven (Dependency Management)

AI/ML: DeepLearning4J, ND4J

Testing: JUnit 5

🏗 Архітектурні особливості
Spatial Optimization: Завдяки просторовому хешуванню складність пошуку об'єктів зменшена з O(N²) до O(1).

Complex Geometry: Підтримка як квадратних, так і трикутних фундаментів з точною математикою з'єднань.

Heuristic Masking: Впроваджено систему маскування невалідних дій для прискорення навчання нейромережі.
